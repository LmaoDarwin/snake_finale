PHYTONS

Game instruction

  1. North with up arrow or ‘w’
  2. South with down arrow or ‘s’
  3. East with right arrow or ‘d’
  4. West with left arrow or ‘a’

The first screen of the game presents the instructions to the player, a text field for player’s name, and the Play buttons for starting the game. The instructions for the game are included in the media files. The “Play” button should be disabled if the text field is empty.
